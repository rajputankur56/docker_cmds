--- Display Docker images
docker images

docker images -q # to return only imageIDs

--- Download docker image from docker hub and run it
docker run <IMAGE>

docker run centos

--- Removing docker images

docker rmi <IMAGEID>

--- Get details of image or container

docker inspect <RepositoryName>
